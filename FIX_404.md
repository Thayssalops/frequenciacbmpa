# Fix: Resolvendo erro 404 no Netlify

## ❌ Problema
Netlify retornava "Page not found" (404) porque:
- Os arquivos HTML estavam em `templates/`
- `netlify.toml` estava configurado para servir `static/`
- Faltava rota padrão para SPA

## ✅ Solução Implementada

### 1. Reorganizar Estrutura
```
ANTES:
├── templates/
│   └── *.html
├── static/
│   └── assets
└── netlify.toml (publish = "static")

DEPOIS:
├── public/              ← Novo diretório de servir
│   ├── *.html
│   ├── static/
│   │   └── assets
│   └── _redirects
├── templates/           ← Mantido para referência
├── static/              ← Mantido para referência
└── netlify.toml (publish = "public")
```

### 2. Arquivos Ajustados

**netlify.toml**
- Mudado: `publish = "public"` (era `"static"`)
- Reordenado: redirects (específico → genérico)
- Criado arquivo `_redirects` como backup

**public/_redirects**
- Novo arquivo com configuração SPA
- Redireciona 404 para `index.html`
